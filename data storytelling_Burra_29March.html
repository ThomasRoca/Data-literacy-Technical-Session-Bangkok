Data Visualization - channels and opportunities

  • Statistical software and different packages/libraries etc.
  • Stata (plugins like Himap), R (packages like ggplot2) and Python (matplolib, bokeh etc.)
  • And now "stuff" is on the cloud, quick and even more interactive, and better suited for data storytelling - Microsoft PowerBI, Carto, Tableau etc.

JavaScript, HTML and CSS form the backbone of modern day data storytelling

  • Hypertext markup language (HTML) is the language of the web - defines content on the web page
  • Cascade style sheet (CSS) defines the design and layout

  • JavaScript programs the behavior (aka interactivity) of the webpage

  • A useful resource always - https://www.w3schools.com/js/ (for JavaScript), https://www.w3schools.com/html/ (for HTML) and https://www.w3schools.com/css/ (for CSS)
